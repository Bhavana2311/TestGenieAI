# 🧠 TestGenie AI

AI Powered Test Case Generation Agent for QA Teams.

## 🚀 Features
- AI Generated Test Cases
- Functional Test Cases
- Negative Test Cases
- Edge Case Detection
- Copy Output
- Download TXT
- Download PDF
- Premium Responsive UI
- Gemini AI Integration

## 🛠️ Tech Stack
- Python
- Flask
- HTML
- CSS
- Bootstrap
- Gemini AI API

## 📌 Problem Statement
Manual test case generation is time-consuming and may miss edge cases. TestGenie AI automates intelligent software testing scenario generation using AI.
